## Understanding and Using APIs in Python

### 1. What is an API?

**API (Application Programming Interface)** is a bridge that allows two software applications to communicate with each other.  
It defines a standard way to **request** and **receive** data securely.

### Example
- When you check weather updates on your phone, your app sends a request to a **weather API**, which returns real-time weather data from its database.
- Similarly, websites like **MakeMyTrip** use **IRCTC’s API** to fetch train data and book tickets without directly accessing IRCTC’s database.

### 2. How APIs Work

APIs exchange data in structured formats — usually **JSON** or **XML**.  
When you send a **request**, the API returns a **response**.

**Basic flow:**
1. Send a request to the API endpoint (URL).  
2. The API processes it and responds with data (JSON).  
3. You extract and use this data in your application.

### 3. Types of API Requests

| Method | Purpose | Example |
|--------|----------|---------|
| **GET** | Retrieve data | Fetch movie details |
| **POST** | Send new data | Add a new user |
| **PUT** | Update existing data | Modify user info |
| **DELETE** | Remove data | Delete an entry |

Most public data APIs use **GET requests**.

### 4. General Steps to Use an API in Python

1. **Find an API** and get an **API key** (if required).  
2. **Check documentation** to find endpoints and parameters.  
3. **Send a request** using the `requests` library.  
4. **Convert the response** (JSON) into Python data.  
5. **Extract and store** the required information (e.g., in a DataFrame).


## 5. General Python Code for Using an API

```python
import requests
import pandas as pd
```

### Step 1: Define the API endpoint
```
url = "https://api.example.com/data"
```

### Step 2: Add parameters or API key if needed
```
params = {"api_key": "YOUR_API_KEY", "query": "example"}
```

### Step 3: Send GET request
```
response = requests.get(url, params=params)
```

### Step 4: Check response status
```
if response.status_code == 200:
    data = response.json()  # Convert to Python dict
else:
    print("Error:", response.status_code)
```
### Step 5: Convert JSON data to DataFrame

```
df = pd.DataFrame(data['results'])  # depends on JSON structure
print(df.head())
```
